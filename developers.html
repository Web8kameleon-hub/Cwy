Authentication

Registration Page

Create account details and complete sign-in with Google Identity Services.

https://cwy.clisonix.com/register.html

GET /api/auth/user

Resolve current user context with a real bearer token or a real session cookie issued by your auth flow.

curl https://cwy.clisonix.com/api/auth/user \
  -H "Authorization: Bearer $AUTH_TOKEN"

POST /api/auth/google/register

Register a user from Google ID token and optional profile details.

curl -X POST https://cwy.clisonix.com/api/auth/google/register \
  -H "Content-Type: application/json" \
  -d '{"idToken":"$GOOGLE_ID_TOKEN","name":"$FULL_NAME"}'

Session Cookie

Cookie-based auth surface using a real session generated by your backend authentication flow.

Cookie: cwy_session=$CWY_SESSION

Export & Chat Constraints

POST /api/v1/export

Supported export formats: PDF, DOCX, XLSX. TXT is rejected. XLSX is relayed to live Excel Core.

curl -X POST https://cwy.clisonix.com/api/v1/export \
  -H "Content-Type: application/json" \
  -d '{"query":"Ops export","format":"xlsx"}'

POST /api/v1/chat

Async processing clamps timeout to 30 seconds maximum. Streaming flag is preserved.

curl -X POST https://cwy.clisonix.com/api/v1/chat \
  -H "Content-Type: application/json" \
  -d '{"message":"Analyze system","timeout":30,"stream":true}'
