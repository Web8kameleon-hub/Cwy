Developer API

API list and live usage model

GET/

Health check for gateway/service state.

curl https://cwy.clisonix.com/

GET/api/cwy/api/stats

License totals, tier stats, and revenue counters.

curl https://cwy.clisonix.com/api/cwy/api/stats

POST/api/validate-license

Validate key online for machine activation.

curl -X POST https://cwy.clisonix.com/api/validate-license -H "Content-Type: application/json" -d '{"license_key":"CWY-PRO-LIFETIME-ABC123"}'

POST/webhook/stripe

Stripe event ingestion that generates licenses after payment.

stripe listen --forward-to https://cwy.clisonix.com/webhook/stripe

WS/ws

Real-time events for dashboard celebrations and activity feed.

wscat -c wss://cwy.clisonix.com/ws

GET/api/cwy

Reserved namespace for CWY REST extensions.

curl https://cwy.clisonix.com/api/cwy
How to Use CWY

From scan to paid activation in three phases

01

Map your system

Run CLI graph analysis and surface cycles, conflicts, and routes.

npx ts-node ./cli/cwy.ts init npx ts-node ./cli/cwy.ts scan npx ts-node ./cli/cwy.ts integrity npx ts-node ./cli/cwy.ts route billing-service
02

Handle payment + licensing

Customer pays, webhook lands, backend generates and verifies license key.

cwy pricing cwy purchase PRO user@example.com cwy verify-payment SESSION_ID cwy activate-license CWY-PRO-LIFETIME-ABC123
03

Unlock feature gates

PRO/ENTERPRISE unlock route, integrity, signals, diff, and export scope.
