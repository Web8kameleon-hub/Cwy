Developer API

CWY API Catalog & Developer Endpoints

Explore health checks, auth, licensing, export workflows, chat processing, WebSocket events, and usage-aware billing surfaces designed for API-first developer teams.

GET/

Health check for gateway/service state.

curl https://cwy.clisonix.com/

GET/api/auth/user

Resolve current user context, active tier, and credit balance.

curl https://cwy.clisonix.com/api/auth/user

GET/api/cwy/api/stats

License totals, tier stats, and revenue counters.

curl https://cwy.clisonix.com/api/cwy/api/stats

POST/api/validate-license

Validate key online for machine activation.

curl -X POST https://cwy.clisonix.com/api/validate-license -H "Content-Type: application/json" -d '{"license_key":"'$CWY_LICENSE_KEY'"}'

POST/webhook/stripe

Stripe event ingestion that generates licenses after payment.

stripe listen --forward-to https://cwy.clisonix.com/webhook/stripe

WS/ws

Real-time events for dashboard celebrations and activity feed.

wscat -c wss://cwy.clisonix.com/ws

GET/api/cwy

Reserved namespace for CWY REST extensions.

curl https://cwy.clisonix.com/api/cwy

POST/api/v1/export

Export report in PDF, DOCX, or XLSX format (not TXT).

curl -X POST https://cwy.clisonix.com/api/v1/export -H "Content-Type: application/json" -d '{"query":"Your analysis","format":"pdf"}'

POST/api/v1/chat

Heavy queries with async timeout (30s). Supports streaming.

curl -X POST https://cwy.clisonix.com/api/v1/chat -H "Content-Type: application/json" -d '{"message":"Analyze system...","timeout":30}'
API Constraints & Best Practices
  • Export formats: PDF, DOCX, XLSX (not TXT)
  • Heavy queries: Async processing with 30s timeout. Use timeout param or fallback to streaming
  • WS events: Subscribe to error, success, progress event types
  • Rate limits: PRO tier → 100 req/min; ENTERPRISE → unlimited
